การติดตั้ง (Installation)
Clone repository นี้โดยใช้คำสั่ง:

bash
คัดลอกโค้ด
git clone https://github.com/fgvfbrgntmyujm/ttkpong.git
เข้าไปที่ไดเรกทอรีของโปรเจค:

bash
คัดลอกโค้ด
cd ttkpong
ติดตั้ง dependencies ที่จำเป็น:

bash
คัดลอกโค้ด
npm install
รันโปรเจค (เช่นสำหรับการพัฒนา):

bash
คัดลอกโค้ด
npm start
